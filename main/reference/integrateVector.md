# Helper for Efficient Integration

Helper for Efficient Integration

## Usage

``` r
integrateVector(integrand, upper, ...)
```

## Arguments

- integrand:

  (`function`)\
  to be integrated.

- upper:

  (`numeric`)\
  upper limits of integration.

- ...:

  additional arguments to be passed to `integrand`.

## Value

This function returns for each upper limit the estimates of the
integral.
